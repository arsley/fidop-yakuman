# 要件

## MUST 絶対欲しい

- 一試合ごとに誰が何点だったかを記録できる
- 試合の記録を一覧で見れる（全体）
- 試合の記録を個別で編集できる
- どこでやったかという開催地を登録できる
- プレイヤー(雀士)を登録できる

## あると良さそう

- 試合のルールを設定して記録できる
  - 三麻・四麻
- ウマ・オカを自由に決められる(表示上で良さそう)
- シーズンの実装?

## ログイン系

- 各々の適当なソーシャルアカウントでSSOできるようにする。
  - 認可のみで良い。
  - 外部サービスのアカウントとプレイヤーの1対1の対応付けはしない。
  - Firebase Authentication で Google account の SSO が使えるようにする想定。
- 全リソースへのアクセス権を制限しない（アプリ内部において）。
